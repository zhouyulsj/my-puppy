请创建一个单文件HTML网页应用，文件名为 index.html。

【技术约束】
- 所有CSS和JS内嵌在HTML中，不依赖本地文件
- 使用 Tailwind CSS CDN（<script src="https://cdn.tailwindcss.com"></script>）
- 使用 Google Fonts 引入 "Noto Sans SC" 中文字体
- 不使用任何框架（React/Vue等），纯原生JS操作DOM
- 代码中所有UI文案使用简体中文

【页面结构】
整体为单页应用，分为两个主视图，通过JS控制显示/隐藏：

视图1 - 宠物档案录入页（初始显示）：
- 顶部：应用名称"🐾 毛球健康管家"，副标题"AI驱动的个性化宠物全周期健康管理"
- 中部：表单卡片，包含以下字段：
  - 宠物名字（text input）
  - 宠物种类（radio：猫/狗，默认选中猫）
  - 品种（select下拉，初始选项为猫的常见品种）
  - 年龄（select下拉：月龄，1-24月，加"2岁""3岁"等成年选项到15岁）
  - 体重（number input，单位kg，step=0.1）
  - 是否绝育（radio：是/否）
- 底部：大按钮"生成健康管家方案"，点击后切换到视图2

视图2 - 方案展示页：
- 顶部栏：宠物信息摘要卡片（名字+品种+月龄+体重+绝育状态），右侧有"更换宠物"按钮回到视图1
- Tab导航栏：4个Tab - "健康评估"、"采购清单"、"使用指南"、"全年日历"
- Tab内容区：目前只放占位文字"正在生成..."，后续步骤填充
- 底部免责声明："本方案由AI生成，仅供参考，不能替代兽医诊断。如宠物出现健康问题请及时就医。"

【样式要求】
- 整体色调：暖色系，主色用 #FF6B35（温暖橙色），辅色 #004E89（深蓝），背景 #FFF8F0（暖白）
- 表单卡片居中，最大宽度 480px，圆角阴影
- Tab栏使用下划线式切换样式，选中态为主色
- 按钮hover有缩放和阴影变化
- 页面加载时有淡入动画
- 响应式：手机端全宽，桌面端居中

【JS功能】
- 宠物种类的radio切换时，品种下拉选项联动更新（猫/狗各准备15个常见品种）
- 点击"生成健康管家方案"时，收集所有表单数据存入全局变量 currentPet，切换到视图2
- 点击"更换宠物"回到视图1，表单保留之前填写的数据
- 所有表单字段必填，提交前校验，未填字段标红提示
