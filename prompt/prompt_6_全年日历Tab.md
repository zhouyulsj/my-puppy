在已有的 index.html 基础上，实现"全年日历"Tab的完整功能。

【功能流程】
1. 用户点击"全年日历"Tab时，如果数据尚未生成，则调用AI生成
2. 生成后缓存到全局变量 calendarData
3. 渲染时间轴视图

【渲染设计 - 垂直时间轴】

整体：左侧一条竖直时间线，右侧按月份分组的事件卡片

每个月份组：
- 左侧时间线上：圆形月份标记，显示"第N月"或月份名
- 右侧：月份标题（如"6月龄（2026年7月）"）
- 下方：该月的事件卡片列表

每个事件卡片：
- 左侧竖条颜色按category：
  - vaccine → 紫色 #8B5CF6
  - deworming → 蓝色 #3B82F6
  - grooming → 绿色 #10B981
  - checkup → 橙色 #F59E0B
  - neutering → 红色 #EF4444
  - special → 粉色 #EC4899
- 右上角importance标记：high→❗ medium→⚠️ low→✓
- 事件标题（加粗）
- 补充说明（灰色小字）
- 右侧复选框"已完成"，勾选后：
  - 事件标题加删除线，整体变半透明
  - 状态存入localStorage（key: furball_calendar_done, value: 事件标识数组）

【顶部统计栏】
- 显示3个数字：总事项数 / 已完成数 / 高优先级未完成数
- 用进度条显示完成比例
